# FULLEMPAQUES Producción

PWA de control de producción para tablets.

## Características

- Login con PIN para operadores
- Vista de órdenes de trabajo por estación
- Cronómetro de trabajo con pausas
- Soporte offline con Service Worker
- Sincronización automática cuando hay conexión

## Tecnologías

- HTML5 / CSS3 / JavaScript
- Supabase (PostgreSQL + Auth + RLS)
- Service Worker para PWA
- IndexedDB para cola offline
